# Zomato Data Analysis Project

## Project Type
Exploratory Data Analysis (EDA) & Regression

## Description
This project analyzes Zomato restaurant metadata to understand pricing patterns
and predict restaurant cost using machine learning models.

## Key Highlights
- Data Cleaning & Wrangling
- 15+ EDA Visualizations (UBM approach)
- Hypothesis Testing
- Feature Engineering
- Multiple Regression Models
- Hyperparameter Tuning
- Model Explainability
- Deployment-ready notebook

## Models Used
- Linear & Ridge Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Final Model
Gradient Boosting Regressor (Optimized)
